ℹ️ About Flixoura

Flixoura is an advanced, feature-rich M3U playlist player designed for the modern streaming era. Built with cutting-edge web technologies, it provides a seamless, beautiful, and powerful interface for managing and watching your favorite content across all devices.

✨ Key Features

  • 🎬 Dual Player System - Switch between Native HTML5 and PlayerJS players
  • 📁 Multi-Source Loading - URL, paste, or drag & drop file upload
  • ⚡ Advanced HLS Streaming - Smooth adaptive bitrate streaming
  • ⭐ Favorites & Bookmarks - Mark your favorite channels
  • 💾 Resume Playback - Pick up exactly where you left off
  • 🔍 Advanced Search & Filters - Find channels instantly
  • 📊 Detailed Statistics - Track your viewing habits
  • 🎨 Dark & Light Themes - Beautiful adaptive themes
  • 📱 Fully Responsive - Perfect on any device
  • ⌨️ Keyboard Shortcuts - Quick navigation
  • 💾 Data Export/Import - Backup your settings

🚀 How to Use

  1. Go to the Player page
  2. Load a playlist from URL, paste M3U content, or upload a file
  3. Browse channels with search and category filters
  4. Click any channel to start streaming
  5. Use the dual player system for best compatibility
  6. Mark favorites and track your watch history
  7. Save playlists to your Library for quick access
  8. View detailed Statistics about your viewing
